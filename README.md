# 🎵 PrivMITLab — Music Player & Live Radio

> **Free, privacy-first music player & live radio hub.** Stream millions of songs in Hindi, Bollywood, Bhojpuri, Punjabi + 50,000+ radio stations worldwide. No account. No tracking. No ads.

🌐 **Live:** [radio-personal.pages.dev](https://radio-personal.pages.dev/)

---

## ✨ What It Does

| Feature | Description |
|---|---|
| 🔍 **Universal Search** | Search any song, artist, or album across multiple providers (Piped, Invidious, YouTube) |
| 📻 **50,000+ Radio Stations** | Browse & stream live radio from every country, filtered by language, genre, and votes |
| 🎤 **40+ Curated Artists** | One-tap access to Arijit Singh, Shreya Ghoshal, AP Dhillon, Pawan Singh, and more |
| 🎭 **Mood-Based Discovery** | Romantic, Workout, Chill, Party, Devotional, Bhojpuri, Punjabi — tap a mood, get music |
| 📴 **Offline Mode** | Cache songs locally via IndexedDB — play without internet |
| ❤️ **Favorites & History** | Save favorites, track recently played — all stored locally |
| 🔀 **Full Player Controls** | Shuffle, repeat (none/one/all), queue management, seek, volume, keyboard shortcuts |
| 🌙 **Dark & Light Mode** | Beautiful glass-morphism UI with dual themes |
| 📱 **PWA Ready** | Install on home screen — works like a native app |
| 🔒 **Zero Tracking** | No accounts, no analytics, no data collection — your music, your privacy |
